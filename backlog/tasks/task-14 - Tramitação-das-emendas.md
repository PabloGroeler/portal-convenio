## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Emenda esta em estatus recebido o setor de orçamento irá colocar ele em analise de admimissibilidade.

Se estiver ok ele envia para analise da demanda para secretaria
caso parecer pela inviabilidade tecnica seja contrario essa emenda ficará com status devolvido.

Secretaria irá colocar em analise de demanda, se estiver ok ele envia para analise documental que será o departamento de convenio.
Caso o parecer pela inviabilidade da demanda for contrario a emenda será devolvida para o setor de orçamento com a justificativa(a justificativa necessita quais pontos não foram atendidos conforme in).

Setor de convenio irá colocar em analise de documentos
se tiver ok ele irá aprovar a emenda e deixar no status iniciado esse status eles irão empenhar a emenda.
Caso o parecer pela reprovação documental for contrario a emenda será devolvida para o setor de orçamento com a justificativa(a justificativa necessita quais pontos não foram atendidos conforme in).

quando a emenda sofrer liquidação o status mudaram para em execução.

quando a emenda sofrer pagamento total ela mudaram para concluido.
