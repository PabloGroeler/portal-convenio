# Application Screens

This document outlines the screens that need to be created for the Mato Grosso Government Portal Clone project.

## 1. Home/Login Page (`index.html`)

*   **Description:** This is the main entry point of the application. It contains the login form and a section for news.
*   **Features:**
    *   User login form.
    *   Link to the registration page.
    *   Display a list of recent news articles.

## 2. Registration Page (`register.html`)

*   **Description:** This page allows new users to register.
*   **Features:**
    *   Registration form with fields for username, password, and email.
    *   Link back to the login page.

## 3. Dashboard Page

*   **Description:** This page is displayed after a successful login.
*   **Features:**
    *   Display the logged-in user's information.
    *   Provide access to the main features of the application, such as:
        *   News management.
        *   User profile management.
        *   Access to other sections of the portal.

## 4. News Page

*   **Description:** This page displays a list of all news articles.
*   **Features:**
    *   List of all news articles with titles and a short summary.
    *   Link to view the full news article.
    *   (Optional) Search and filter functionality.

## 5. News Article Page

*   **Description:** This page displays a single news article.
*   **Features:**
    *   Display the full content of the news article.
    *   (Optional) Comment section.

## 6. Contact Page

*   **Description:** This page displays contact information.
*   **Features:**
    *   Contact form.
    *   Address and phone number.
    *   (Optional) Map with the location.

## 7. Manuals Page

*   **Description:** This page lists and provides links to download manuals.
*   **Features:**
    *   List of available manuals.
    *   Links to download the manuals in PDF or other formats.

## 8. Downloads Page

*   **Description:** This page lists and provides links to download other files.
*   **Features:**
    *   List of available files for download.
    *   Links to download the files.

## 9. Legislation Page

*   **Description:** This page displays legislation information.
*   **Features:**
    *   List of laws and regulations.
    *   Links to view the full text of the legislation.
